# 🎯 Personal Introduction Program

This is a simple Python program that asks the user for basic information and displays a friendly welcome message.

## 🛠️ Features
- Takes user input using `input()`
- Stores data in variables
- Uses formatted strings (f-strings)
- Displays a personalized message

## 📋 Questions Asked
- Name
- Age
- Hobby
- City

## ▶️ How to Run

1. Install Python
2. Clone this repository:
   ```bash
   git clone https://github.com/your-username/personal-intro.git
3. Run the program: python personal_intro.py
