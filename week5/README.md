# Customer Sales Analysis

## Project Overview

Customer Sales Analysis is an advanced data analysis project developed using Python, Pandas, and Matplotlib.

The project analyzes customer purchasing behavior, sales performance, product revenue, and regional sales distribution. It demonstrates advanced Pandas operations such as grouping, aggregation, filtering, datetime manipulation, pivot tables, and data visualization.

---

## Features

- Load and analyze sales and customer datasets
- Handle missing values and duplicate records
- Extract year, month, and day from date columns
- Group and aggregate sales data
- Identify top customers
- Analyze product performance
- Analyze regional sales distribution
- Create pivot tables
- Generate professional visualizations
- Export charts automatically

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Customer-Sales-Analysis.git
```

### Open Project

```bash
cd Customer-Sales-Analysis
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python customer_analysis.py
```
