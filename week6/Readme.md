# Interactive Sales Dashboard

## Project Overview

This project is an advanced data visualization and dashboard creation project developed as part of my internship tasks at The Developer's Arena.

The objective is to transform raw sales data into meaningful visual insights using Seaborn, Matplotlib, and Plotly. The dashboard combines multiple chart types to analyze sales performance, product trends, regional distribution, and statistical relationships.

---

## Objectives

- Learn advanced data visualization
- Create professional dashboards
- Use Seaborn statistical plots
- Build correlation heatmaps
- Create interactive Plotly charts
- Present business insights visually

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Git
- GitHub

---

## Features

- Product Revenue Analysis
- Regional Sales Analysis
- Sales Distribution Analysis
- Correlation Analysis
- Monthly Trend Analysis
- Interactive Dashboard

---

## Visualizations Included

### Revenue by Product
Bar chart showing product performance.

### Regional Sales Distribution
Pie chart showing region contribution.

### Product Sales Distribution
Box plot for understanding sales spread.

### Correlation Heatmap
Displays relationships between numeric variables.

### Monthly Trend Analysis
Line chart showing revenue trends.

### Interactive Plotly Dashboard
Interactive chart with hover effects.
