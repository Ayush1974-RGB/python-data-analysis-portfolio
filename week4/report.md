# E-Commerce Sales Analysis Report

## Introduction

The objective of this project is to analyze sales performance using
Python and data visualization techniques.

## Dataset Description

The dataset contains 100 sales transactions with information about:

- Product
- Quantity
- Price
- Region
- Sales Amount
- Date

## Methodology

1. Loaded dataset using pandas
2. Cleaned missing values and duplicates
3. Performed exploratory analysis
4. Calculated sales metrics
5. Created visualizations using matplotlib

## Results

### Top Product

Laptop generated the highest revenue.

### Regional Performance

North region showed the largest number of transactions.

### Revenue Analysis

Total Revenue:
₹12,365,048

Average Revenue Per Transaction:
₹123,650

## Conclusion

The analysis shows that Laptop and Tablet categories are major revenue
drivers. Businesses should focus marketing efforts on these products
while improving sales performance in lower-performing regions.
