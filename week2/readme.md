# Student Grade Calculator

## Project Overview
This is a simple Python project that calculates student grades based on marks entered by the user.

The program:
- Takes student name and marks as input
- Validates marks between 0 and 100
- Calculates grades using if-elif-else statements
- Displays encouraging messages
- Handles invalid input using loops and error handling

This project helps beginners understand:
- Conditions
- Loops
- Functions
- Input validation
- Error handling
